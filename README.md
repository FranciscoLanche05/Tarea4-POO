Deber 4: Programación Orientada a Objetos en Java

📖 Descripción

Este proyecto contiene la resolución de los ejercicios prácticos del Deber 1 para la asignatura de Programación Orientada a Objetos (POO) de la Escuela Politécnica Nacional (EPN).

El objetivo principal es aplicar los conceptos fundamentales de Java, enfocándose en la estructuración de clases, manipulación de datos por consola y protección de la información.

🚀 Conceptos Aplicados

Lectura de datos: Uso de la clase Scanner.

Tipos de Métodos: - Métodos void (ejecución de acciones, sin retorno).

Métodos con return (cálculos lógicos y matemáticos).

Constructores: Inicialización de objetos.

Encapsulamiento: Modificadores de acceso (private, public), y uso de métodos get y set aplicando validaciones y reglas de negocio.

📂 Ejercicios Desarrollados

Control de Estudiante: Implementación básica de métodos void para entrada y salida de datos.

Control de Producto: Cálculo de totales utilizando únicamente métodos void.

Cálculo de Promedio Final: Retorno de valores numéricos y cadenas de texto mediante return.

Planilla de Luz: Cálculos encadenados de subtotal, IVA y total.

Control de Empleado: Uso mixto de métodos void y return para separar la lógica de presentación.

Pedido en Restaurante: Aplicación de condiciones de descuento y cálculos de facturación.

Control de Inventario y Reposición: Ejercicio integrador con encapsulamiento estricto, validación para evitar stocks o precios negativos, y evaluación automática del estado del producto ("Stock suficiente", "Requiere reposición" o "Producto agotado").

🛠️ Tecnologías

Lenguaje: Java (JDK 8 o superior).

Entorno recomendado: IntelliJ IDEA, Eclipse, o Apache NetBeans.

⚙️ Cómo ejecutar

Clona este repositorio o descarga el código fuente.

Abre la carpeta del proyecto en tu IDE favorito.

Navega hacia el paquete del ejercicio que deseas probar (por ejemplo, Ejercicio7).

Haz clic derecho sobre el archivo Main.java de ese paquete y selecciona Run.

Sigue las instrucciones que aparecerán en la consola.
